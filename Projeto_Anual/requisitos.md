# **Projeto Anual - Requisitos**
## **Plataforma de Venda de Jogos**

<br />
O objetivo do projeto é de desenvolver uma plataforma de venda de jogos, como os seguintes sites:

* [Steam](https://store.steampowered.com/?l=portuguese)
* [Nuuvem](https://www.nuuvem.com) 
* [Origin](https://www.origin.com/bra/pt-br/store)

### **O que nossa plataforma precisa ter?**
A priori, as seguintes funcionalidades:
1. Venda de Jogos
2. Promoções
3. Ferramenta de Busca
4. Categorias de Jogos
5. Compra, Devolução, Reembolso
6. Feedback
7. Perfil de cada Jogo
8. Sistema de Troféus
9. Conta do usuário

### **O que o usuário poderá fazer?**
Poderá comprar novos jogos ao longo da utilização da plataforma. Caso ele se arrependa, poderá devolver e pedir reembolso (ou receber um saldo para comprar outro jogo). O prazo máximo de arrependimento será de **7 dias**.

Além disso, poderá dar **feedbacks** sobre os jogos que ele **possui**.

Cada jogo terá um perfil, com dados sobre ele e um sistema de troféus.
